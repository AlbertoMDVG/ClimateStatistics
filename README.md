# ClimateStatistics
Some ideas and coursework related to the module Climate statistics from MSc Statistical Science Oxford.
Contains 
- some climate data visualization
- Smoothing, detrending and general data preprocessing
- Time series
- Extreme value theory
- Multivariate distributions

The file main.ipynb is where things are put into context and roughly explain about what I am trying to do
There are auxiliary files like web scrappers of other functions and classes that would obstruct the main 
focus of the course.
